# Kodify Chat

## Setup

1. Install create-react-app
```
npm install -g create-react-app
```

2. Install packages
```
npm install express --save
npm install socket.io --save
```

## Start the server

This will start the server on port 8890
```
node server.js
```

## Start the client

This will start the web app client on port 3000

```
npm start
```
